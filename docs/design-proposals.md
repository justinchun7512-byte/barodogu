## 안 1: "조판실" (Typesetting Room)

### 컨셉
취업 서류를 다루는 도구 사이트답게, **인쇄소·편집실 미학**을 현대적으로 재해석한다.
격자(grid), 기준선(baseline), 여백의 규칙성을 전면에 드러내는 디자인.
"도구"를 쓰는 느낌이 아니라 **작업실에 들어온 느낌**.
일본 그래픽 디자인과 스위스 타이포그래피의 교차점.

### 색상 팔레트
```
배경 (라이트): #F5F2EC  — 누런 크림, 신문 용지 느낌
배경 (다크):   #1A1814  — 인쇄 잉크 블랙
Primary:       #C8401A  — 교정 빨간 펜, 도장 빨간색
Secondary:     #2B5EA7  — 공문서 파란 잉크
Accent:        #E8D5A3  — 노란 포스트잇
텍스트:        #1A1814 / #F5F2EC
보조 텍스트:   #6B6560 / #9B9590
경계선:        #D4CFC8 / #2E2A26
```

### 타이포그래피
- **헤드라인**: `Nanum Myeongjo` (Google Fonts) — 명조체의 권위감, 취업 서류 느낌
- **서브헤드**: `Nanum Myeongjo` Bold + 자간 -0.02em
- **본문/UI**: `IBM Plex Sans KR` (Google Fonts) — 기술적·중립적 고딕
- **숫자/코드**: `IBM Plex Mono` — 계산기 숫자에 적합
- 폰트 스케일: 12 / 14 / 16 / 20 / 28 / 40 / 56px (4px 기반)

### 레이아웃 특징
1. **격자선 노출**: 페이지 배경에 연한 수직선 그리드 (신문 조판 느낌)
2. **비대칭 히어로**: 왼쪽 정렬 헤드라인 + 오른쪽에 도구 숫자 카운터 (ex: "31개의 도구")
3. **도구 카드**: 흰 박스 대신 테두리선만 있는 카드 (outline only, 배경 없음)
4. **카테고리**: 알약 버튼 대신 **가로 탭 + 밑줄 강조** (신문 섹션 구분 방식)
5. **HOT/NEW 뱃지**: 빨간 스탬프 형태 (rotated, slightly imperfect)
6. **검색창**: 중앙 상단 고정이 아니라, 페이지 타이틀 바로 아래 풀와이드 라인 스타일
7. **푸터**: 신문 하단 크레딧 스타일 — 세로 구분선 + 소형 텍스트 정렬

### 핵심 차별점
- 한국 취업 사이트 중 **명조체 헤드라인 + 편집 디자인 미학** 사용 사례 없음
- "도구" 느낌이 아닌 **전문 작업실** 느낌으로 신뢰도 상승
- 색상이 차갑지 않고 **크림+레드**로 따뜻한 권위감
- SaaS 파란색 그라데이션에서 완전히 탈피

### 참고 레퍼런스
- `fonts.google.com` → Nanum Myeongjo 적용 예시
- 한국일보·조선일보 디지털 에디션 레이아웃
- Muji 브랜드 디자인 (여백, 크림색, 미니멀)
- 일본 디자인 잡지 `Idea` (조판 미학)
- `arc.net` 브라우저 사이트 (비대칭 레이아웃)

## 안 2: "터미널 도구" (Terminal Tools)

### 컨셉
취업 준비를 **작전처럼** 수행하는 느낌. 개발자 감성이지만 무섭지 않고,
CLI(커맨드라인)에서 모티프를 따온 **인터랙티브·기능적** 미학.
"도구"라는 단어를 있는 그대로 시각화. 사용자가 "뭔가 강력한 걸 쓰는 것 같은" 느낌.
Vercel · Linear · Raycast 의 다크 퍼스트 접근법.

### 색상 팔레트
```
배경 (라이트): #FAFAFA  — 거의 흰색, 매우 밝음
배경 (다크):   #0D0D0D  — 터미널 블랙
Primary:       #00D084  — 터미널 그린 (성공/완료)
Secondary:     #FF6B35  — 경고/강조 오렌지
Accent:        #A855F7  — 보라 (AI 카테고리 한정)
텍스트:        #111111 / #E8E8E8
보조 텍스트:   #666666 / #888888
경계선:        #E5E5E5 / #222222
코드 블록 BG: #F4F4F4 / #161616
```

### 타이포그래피
- **헤드라인**: `Space Grotesk` (Google Fonts) — 기하학적 고딕, 개성 강함
- **서브/본문**: `Pretendard Variable` — 한글 가독성 유지
- **숫자·태그**: `JetBrains Mono` — 코드 폰트로 기능성 강조
- 특이점: 헤드라인에 `font-feature-settings: "ss01"` 대체 글리프 활용

### 레이아웃 특징
1. **히어로 = 가짜 터미널 창**: 애니메이션 타이핑 효과로 `$ barodogu --find "연봉계산기"` 표시 후 결과 출력
2. **카드 리스트뷰 기본**: 3열 그리드 대신 **2열 리스트** (이름 + 설명 + 카테고리 태그 한 줄)
3. **카테고리 = 해시태그 필터**: `#취업` `#AI` `#금융` 형태의 코드 스타일 태그
4. **로고**: 바로도구를 `baro://` URL scheme처럼 표현
5. **검색**: 커맨드팔레트 스타일 (Cmd+K 단축키 지원, 중앙 모달 팝업)
6. **도구 카드 호버**: 왼쪽에 초록 `>` 커서 나타남 (터미널 선택 느낌)
7. **뱃지 대신 접두사**: `[HOT]` `[NEW]` `[AI]` 텍스트 스타일 레이블

### 핵심 차별점
- 취업 사이트지만 **개발자 감성**을 빌려와 "스마트한 취준생" 포지셔닝
- 커맨드팔레트 검색은 현재 어떤 국내 유틸 사이트도 없음
- 다크모드가 라이트모드보다 더 자연스러운 설계 (다크 퍼스트)
- `#태그` 카테고리 필터로 SNS에 익숙한 20대에게 친숙

### 참고 레퍼런스
- `linear.app` — 리스트뷰 + 다크 퍼스트 + 모노 타입
- `raycast.com` — 커맨드팔레트 중심 UX
- `vercel.com` — 블랙 배경 + 그린 액센트
- `fig.io` — 터미널 미학의 현대화

## 안 3: "도구 잡지" (Tool Magazine)

### 컨셉
취업 준비를 **연구하듯** 접근하는 사용자를 위한 **큐레이션 매거진** 스타일.
도구들이 단순 나열이 아니라 **편집된 콘텐츠**처럼 보인다.
무인양품 카탈로그 + 네이버 시리즈 + 디자인 매거진.
"발견"의 즐거움이 있는 구조.

### 색상 팔레트
```
배경 (라이트): #FFFFFF  — 순백
배경 (다크):   #141414  — 깊은 검정
Primary:       #FF3B3B  — 잡지 레드 (강렬, 주목)
Secondary:     #1A1A2E  — 딥 네이비 (권위)
Accent:        #F0E6FF  — 연보라 (AI 섹션)
Highlight:     #FFF176  — 형광 노랑 (중요 강조)
텍스트:        #141414 / #F5F5F5
보조 텍스트:   #757575 / #AAAAAA
경계선:        #EBEBEB / #2A2A2A
```

### 타이포그래피
- **대형 헤드라인**: `Bebas Neue` (Google Fonts) — 잡지 표지 대문자 타이틀
- **한글 헤드**: `Noto Serif KR` Bold — 신뢰감 있는 명조
- **본문**: `Noto Sans KR` Regular — 가독성 최우선
- **강조 숫자**: `Oswald` Bold — 큰 숫자 표현에 최적
- 레이아웃 핵심: 문자 크기 대비 극대화 (6px ~ 72px 대비)

### 레이아웃 특징
1. **불규칙 히어로 그리드**: "BARO DOGU" 대문자 + 왼쪽에 도구 카드 콜라주 배치
2. **편집장 추천 섹션**: 3×1 대형 카드 + 작은 카드 6개 혼합 (잡지 레이아웃)
3. **카테고리 = 섹션 타이틀**: 잡지의 Chapter 구분처럼 큰 섹션 제목으로 구분
4. **도구 카드 2가지 사이즈**: Feature Card (가로 전체) + Regular Card (1/3) 혼합
5. **형광 하이라이트**: 중요 텍스트에 배경 형광 마커 효과
6. **스크롤 스토리텔링**: 각 카테고리 섹션에 짧은 카피 1줄씩
7. **검색창 스타일**: 잡지 검색란처럼 밑줄만 있는 언더라인 인풋

### 핵심 차별점
- **Bento Grid 대신 Editorial Grid** — 잡지 레이아웃으로 시각적 흥미
- 도구를 "쓰는" 것이 아니라 "발견"하게 만드는 UX
- 형광 노랑 + 잡지 레드 조합은 한국 유틸 사이트에서 전혀 없는 조합
- 비균일 카드 크기로 중요 도구(HOT)를 자연스럽게 강조

### 참고 레퍼런스
- `notion.so` 2023 리뉴얼 (에디토리얼 히어로)
- `stripe.com` (타이포그래피 대비)
- 잡지 `Monocle`, `Kinfolk` (레이아웃 그리드)
- `every.to` (큐레이션 매거진 + 도구 혼합 사이트)
- `a24films.com` (대문자 타이포그래피 미학)

## 구현 우선순위 권고

**1순위 구현 추천: 안 2 (터미널 도구)**
- 이유: 커맨드팔레트 검색이 바로도구 UX의 핵심 기능이 될 수 있음
- 국내 유사 사이트 전혀 없는 완전 차별화
- 다크모드 완성도 가장 높음
- Space Grotesk + Pretendard 조합이 기술적으로 구현 쉬움

**2순위: 안 1 (조판실)**
- 이유: 취업 포지셔닝과 직접 연결되는 테마
- 명조체 + 크림 배경이 신뢰도 상승에 가장 효과적
- 구현 복잡도 낮아 빠른 출시 가능

**3순위: 안 3 (도구 잡지)**
- 이유: 도구 수가 늘어날수록 더 빛나는 구조
- 현재 30개 수준에서는 에디토리얼 그리드 채우기 어려울 수 있음
- 추후 50개+ 도달 시 전환 검토

## 공통 개선 제안 (어느 안을 선택하든)

1. **로고 교체**: "B" 박스 로고 → 워드마크 형태 (텍스트 로고)
2. **폰트 교체**: Pretendard 단독 → 선택한 안의 헤드라인 폰트 추가
3. **Primary 컬러 교체**: `#2563EB` 탈피 → 안에 맞는 독자적 컬러
4. **히어로 구조 해체**: 중앙정렬 텍스트 + 체크마크 3개 → 각 안의 구조로
5. **카드 배경색 교체**: 흰 카드 그리드 탈피
