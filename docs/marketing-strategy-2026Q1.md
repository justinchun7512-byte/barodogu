## 3. 쇼츠/릴스 콘텐츠 계획

### 3-1. 바이럴 가능성 높은 주제 TOP 5

**1. "자소서 핵심역량, AI가 30초 만에 뽑아줌" (바이럴 잠재력: 최상)**
- 포맷: 화면 녹화 + 자막
- 스크립트 방향: "채용공고 붙여넣기 -> 이력서 붙여넣기 -> AI가 핵심역량 3개 추출 -> 실제 자소서에 적용"
- 후킹: "자소서 핵심역량 때문에 밤새는 거 이제 그만"
- 길이: 30-45초

**2. "연봉 5000이면 실수령 얼마일까?" (바이럴 잠재력: 상)**
- 포맷: 숫자 입력하는 장면 + 결과 강조
- 스크립트 방향: "연봉 3000/4000/5000/6000 실수령액 비교" 빠른 전환
- 후킹: "연봉 5000인데 통장에 꽂히는 돈은..."
- 길이: 20-30초

**3. "퇴직금 못 받는 3가지 경우" (바이럴 잠재력: 상)**
- 포맷: 텍스트 오버레이 + 계산기 사용 장면
- 스크립트 방향: 충격적 사실 -> 해결책(퇴직금 계산기로 미리 확인)
- 후킹: "회사 3년 다녔는데 퇴직금이 0원?"
- 길이: 30-40초

**4. "맞춤법 하나 틀렸다고 자소서 불합격?" (바이럴 잠재력: 중상)**
- 포맷: 실제 틀리기 쉬운 맞춤법 퀴즈 + 검사기 시연
- 스크립트 방향: "인사담당자가 가장 싫어하는 맞춤법 실수 TOP 5"
- 후킹: "이거 틀리면 서류 탈락 확정"
- 길이: 40-50초

**5. "취준생이 모르면 손해인 무료 사이트" (바이럴 잠재력: 중상)**
- 포맷: 빠른 사이트 전환 + 각 도구 1줄 설명
- 스크립트 방향: 바로도구 13가지 도구를 3초씩 보여주기
- 후킹: "취준생한테만 알려주는 무료 사이트"
- 길이: 45-60초

### 3-2. 쇼츠 운영 전략

- **플랫폼**: YouTube Shorts > Instagram Reels > TikTok (동시 업로드)
- **발행 주기**: 주 3회 이상

## 4. 커뮤니티 공유 전략

### 4-1. 타깃 커뮤니티 및 접근법

#### 에브리타임 (대학생/취준생 핵심 채널)
- **게시판**: 자유게시판, 취업게시판, 홍보게시판
- **공유 형식**:
  - (자유게시판) "공채 시즌인데 자소서 핵심역량 뽑기 너무 어렵지 않나요? 저는 이 사이트 써봤는데 채용공고 넣으면 자동으로 뽑아줌. 무료임"
  - (취업게시판) "자소서 핵심역량 작성할 때 유용한 무료 도구 공유합니다 (광고 아님, 실사용 후기)"
- **주의**: 광고성 글은 삭제될 수 있으므로, 실제 사용 후기 형식으로 작성. 스크린샷 첨부.

#### 블라인드 (직장인/이직 준비자)
- **게시판**: 이직/퇴사, 자유게시판
- **공유 형식**:
  - "이직 준비 중인데 채용공고에서 핵심역량 뽑아주는 사이트 발견함. JD 넣으면 바로 분석해줌"
  - "퇴직금 계산해봤더니 생각보다 적어서 충격... 여기서 계산해보세요"
- **주의**: 블라인드는 회사 이메일 인증 유저만 가능, 자연스러운 경험 공유 톤 유지

#### 디시인사이드 (취업 갤러리)
- **공유 형식**: "자소서 핵심역량 뽑는 거 AI한테 시킴 ㅋㅋ 생각보다 괜찮음"
- **톤**: 디시 문화에 맞게 캐주얼하게, 스크린샷 필수
- **주의**: 너무 정돈된 글은 오히려 광고 의심 받음

#### 클리앙 (IT/개발자)
- **게시판**: 새로운소식, 모두의공원
- **공유 형식**: "Next.js로 만든 무료 유틸리티 사이트 공유 (개발 후기)"
- **톤**: 기술적 관점에서 접근, 개발 과정 공유

#### 네이버 카페 (취업 관련)
- **타깃 카페**: 취뽀(취업에 뽀개기), 스펙업, 취업대학교 등
- **공유 형식**: "상반기 공채 준비하면서 유용했던 무료 도구 모음"
- **핵심**: 정보성 글 형태, 여러 도구를 소개하면서 바로도구를 자연스럽게 포함

#### 레딧/오픈카톡방
- **레딧 r/korea, r/hanguk**: 영문 또는 한영 혼합 소개
- **카카오 오픈채팅**: 취준생 오픈카톡방에 유용한 정보로 공유

### 4-2. 자연스러운 공유 원칙

1. **"발견" 형태로 작성**: "이런 거 찾았는데..." / "써봤더니 괜찮아서 공유"
2. **실제 사용 스크린샷 첨부**: 글만 있으면 광고처럼 보임
3. **단점도 함께 언급**: "도구 수가 아직 적긴 한데, 핵심역량 추출기는 진짜 쓸만함"
4. **댓글에서 추가 정보 제공**: 본문은 짧게, 댓글에서 링크 공유
5. **하루에 1개 커뮤니티만**: 동시 다발적 게시는 스팸으로 인식됨
6. **계정 관리**: 오래된 계정 사용, 다른 글에도 평소처럼 활동

## 7. 리스크 및 기회

### 리스크
1. **SEO 반영 지연**: 신규 도메인은 구글 인덱싱에 2-4주 소요 -> 커뮤니티 마케팅으로 초기 트래픽 확보
2. **콘텐츠 생산 부담**: 1인 운영으로 주 4-5개 글 작성 부담 -> AI 도구(Claude) 활용하여 초안 작성 효율화
3. **커뮤니티 삭제 리스크**: 광고성 글 삭제 가능 -> 자연스러운 후기 형태 + 분산 게시
4. **경쟁사 대응**: 모두의도구(116개 도구) 대비 도구 수 열위 -> "취업 특화" 차별점으로 승부

### 기회
1. **공채 시즌 타이밍**: 3-4월 상반기 공채 피크 시즌, 핵심역량 관련 검색량 급증 예상
2. **AI 차별화**: 경쟁 취업 사이트에 없는 "AI 핵심역량 추출" 기능은 독보적 차별점
3. **직무 중심 채용 트렌드**: 2026년 채용은 직무역량 중심 -> 핵심역량 추출 도구 수요 증가
4. **무료 도구 바이럴**: 유용한 무료 도구는 자발적 공유 가능성 높음
