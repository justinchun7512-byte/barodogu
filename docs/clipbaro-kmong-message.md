# 클립바로 크몽 구매 완료 메시지 템플릿

크몽에서 구매 완료 후 구매자에게 발송하는 메시지입니다.
판매자 관리 > 자동 메시지 또는 직접 발송에 사용하세요.

---

## 메시지 본문

```
안녕하세요! 클립바로를 구매해주셔서 감사합니다 🎉

▶ 라이선스 발급 방법 (1분이면 완료!)

1. 아래 링크에 접속하세요:
   👉 https://barodogu.com/clipbaro/activate

2. 크몽 주문번호와 이메일을 입력하세요

3. 발급된 라이선스 키를 클립바로 앱에 입력하면 끝!

▶ 클립바로 다운로드
   👉 (다운로드 링크)

문의사항이 있으시면 언제든 크몽 메시지로 연락주세요!
```

---

## 패키지별 안내 메모

| 크몽 패키지 | 내부 등급 | 가격 | 활성화 페이지에서 선택 |
|-----------|---------|------|-----------------|
| STANDARD  | basic   | 9,900원 | 기본 (STANDARD) |
| DELUXE    | starter | 15,000원 | 스타터 (DELUXE) |
| PREMIUM   | pro     | 20,000원 | 프로 (PREMIUM) |

---

## 주의사항

- 구매자가 활성화 페이지에서 주문번호를 잘못 입력하면 다른 키가 발급됩니다.
- 주문번호는 크몽 '내 거래 > 주문 상세'에서 확인하도록 안내하세요.
- 같은 주문번호로 재발급 시 항상 동일한 키가 나옵니다 (서버리스 결정론적 생성).
- Vercel 환경변수 CLIPBARO_LICENSE_SECRET이 설정되어 있어야 합니다.
