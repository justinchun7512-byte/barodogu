# 클립바로 크몽 구매 완료 메시지 템플릿

크몽에서 구매 완료 후 구매자에게 발송하는 메시지입니다.
판매자 관리 > 자동 메시지 또는 직접 발송에 사용하세요.

---

## 메시지 본문

```
안녕하세요! 클립바로를 구매해주셔서 감사합니다 🎉

▶ 설치 방법 (5분이면 완료!)

1. 아래 링크에서 클립바로를 다운로드하세요:
   👉 (다운로드 링크)

2. 압축을 풀고 ClipBaro 파일을 실행하세요

   ⚠️ Mac 사용자: 처음 실행 시 보안 경고가 뜰 수 있습니다.
   아래 방법 중 하나로 해결하세요:
   
   [방법 1] 터미널을 열고 아래 명령어를 복사해서 실행하세요:
   xattr -cr ~/다운로드/clipbaro/
   그 후 ClipBaro를 다시 더블클릭하세요.
   
   [방법 2] 시스템 설정 > 개인정보 보호 및 보안 > 하단에서
   "ClipBaro 확인 없이 열기"를 클릭하세요.

   ⚠️ Windows 사용자: "Windows가 PC를 보호했습니다" 경고 시
   "추가 정보" > "실행" 을 클릭하세요.

3. 브라우저가 자동으로 열리면 설치 완료!

▶ 사용법 영상 (처음부터 끝까지 4분이면 이해 가능!)
   👉 https://youtu.be/3nfXjbKuaYI

▶ 라이선스 발급 방법 (1분이면 완료!)

1. 아래 링크에 접속하세요:
   👉 https://barodogu.com/clipbaro/activate

2. 크몽 주문번호와 이메일을 입력하세요
   (주문번호: 크몽 > 내 거래 > 주문 상세에서 확인)

3. 발급된 라이선스 키를 클립바로 앱 > 설정 > 라이선스에 입력하면 끝!

▶ 사전 준비 (필수)

FFmpeg가 설치되어 있어야 영상이 만들어집니다:
- Mac: 터미널에서 brew install ffmpeg
- Windows: 시작 메뉴 > cmd > winget install ffmpeg

▶ API 키 등록 (필수)

클립바로 앱 > 설정 페이지에서 Gemini API 키를 등록하세요:
- Gemini API 키 발급 (무료): https://aistudio.google.com/apikey

문의사항이 있으시면 언제든 크몽 메시지로 연락주세요!
```

## 패키지별 안내 메모

| 크몽 패키지 | 내부 등급 | 가격 | 활성화 페이지에서 선택 |
|-----------|---------|------|-----------------|
| STANDARD  | basic   | 9,900원 | 기본 (STANDARD) |
| DELUXE    | starter | 15,000원 | 스타터 (DELUXE) |
| PREMIUM   | pro     | 20,000원 | 프로 (PREMIUM) |

---

## 주의사항

- 구매자가 활성화 페이지에서 주문번호를 잘못 입력하면 다른 키가 발급됩니다.
- 주문번호는 크몽 '내 거래 > 주문 상세'에서 확인하도록 안내하세요.
- 같은 주문번호로 재발급 시 항상 동일한 키가 나옵니다 (서버리스 결정론적 생성).
- Vercel 환경변수 CLIPBARO_LICENSE_SECRET이 설정되어 있어야 합니다.
